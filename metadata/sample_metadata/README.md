This is the final metadata that I am using and the following columns are important to know: 

Sample_ID= Sanger ID 
Organ= site of sampling 
10X_Chemistry= 10X kit version 
Method: 5prime versus 3prime versus multiome 
Technology: Cellular versus nuclear RNA 
Age: Age of samples
Multiplexed: Yes/No
Study: Name of publication associated with data 
Accession_ID: raw data location 
FACS_Sorting: are samples sorted
