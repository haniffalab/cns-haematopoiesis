# metadata/

Contains all metadata tables and results:
- **marker_genes/**: gene lists/notebook with marker genes used for annotation
- **old_annotations/**:  cell-type labels for the data that are published, processed datasets with annotations as csvs
- **supercell_results/**: aggregated super-cell outputs
- **sample_metada/**: contains the metadata files as csv 
- **Metadata_Cleanup.ipynb**: contains the information about how metadata file was created 
- **CellTypist_Hierarchy_Levels3_2_1.csv**: the annotations from celltypist model trained on developmental datasets. The annotations are categorised into levels 1, 2 and 3 where level 3 are 536 annotations from the model
